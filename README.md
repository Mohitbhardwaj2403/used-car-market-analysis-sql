# 🚗 Exploring Trends in the Automotive Industry Using SQL

## 📌 Project Overview

This project analyzes the used automobile market using SQL to identify pricing trends, mileage patterns, ownership impact, fuel preferences, and market behavior.

The analysis is performed on a dataset containing over **8,148 vehicle records**, demonstrating how SQL can transform raw data into meaningful business insights.

---

## 🎯 Objectives

- Analyze vehicle pricing trends
- Compare fuel types and transmission categories
- Study ownership impact on resale value
- Evaluate mileage and engine performance
- Perform advanced SQL analysis using CTEs and Window Functions
- Generate actionable business insights

## 📊 Key Insights

- Luxury brands have the highest resale values.
- Newer vehicles command significantly higher prices.
- Automatic transmission vehicles are generally more expensive.
- First-owner vehicles retain higher resale value.
- Diesel vehicles provide better average mileage.
- Vehicle age strongly impacts depreciation.
- Petrol and Diesel dominate the used car market.
